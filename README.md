# -Crime-Data-Analysis-
A beginner-friendly Python project that analyzes crime data using Pandas, Matplotlib, and SQLite. It performs data cleaning, generates visualizations, stores records in a database, and exports reports for analysis.
Features

Load crime data from a CSV file:-
-Remove duplicate records
-Handle missing values
-Extract year information from the Date column
-Display city-wise crime statistics
-Display crime-type distribution
-Store processed records in an SQLite database
-Generate bar charts using Matplotlib
-Generate pie charts for crime type distribution
-Export processed data to crime_report.csv

language and library Used:-

-Python
-Pandas
-Matplotlib
-SQLite3

Dataset Information The dataset contains the following fields:-

-Crime_ID
-City
-State
-Crime_Type
-Date
-Victim_Age
-Victim_Gender
-Status
-Police_Station
-Officer_Assigned
-Case_Priority
-Estimated_Loss_INR

Program Output The project performs the following tasks:-

--Reads the crime dataset
-Displays the first few records
-Cleans duplicate and missing data
-Counts crimes by city
-Counts crimes by type
-Saves records in SQLite
-Generates visual charts
-Creates a processed report (crime_report.csv)
